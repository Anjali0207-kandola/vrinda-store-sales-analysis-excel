# 📊 Vrinda Store Sales Dashboard (Excel)

## 📌 About the Project
This project analyzes Vrinda Store sales data using Microsoft Excel. An interactive dashboard was created using Pivot Tables, charts, and slicers to understand sales performance and customer behavior.

---

## 🎯 Objective
To analyze sales data and identify meaningful insights that can help improve business decision-making.

---

## 🛠️ Tools Used
- Microsoft Excel  
- Pivot Tables  
- Charts  
- Slicers  

---

## ❓ Key Questions
- Which state has the highest sales?  
- Which gender contributes more to sales?  
- Which age group buys the most products?  
- Which sales channel performs best?  

---

## 📈 Key Insights
- Women customers contributed more to overall sales  
- Maharashtra and Karnataka are the top-performing states  
- Adult age group (30–49) generates the highest orders  
- Amazon and Flipkart are the leading sales channels
